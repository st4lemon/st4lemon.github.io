Projects

Cflat Compiler | C++, x86-64 | CS160

Built a compiler from scratch for a simplified C-style programming language.

  • - Implemented a longest-match algorithm for lexing and tokenizing programs
  • - Created a parser using an LL(1) context-free grammar to translate tokens into an abstract syntax tree
  • - Wrote code to translate a lower-intermediate representation into x86-64 assembly code
  • - Large project: over 5000 lines of code

Pipelined MIPS CPU Design | Python, PyRTL | CS154

CPU design supporting MIPS architecture

  • - Split instruction execution into 5 pipelined cycles
  • - Implemented forwarding and stalling to improve performance
  • - Created a hierarchical page table and round-robin replacement cache for fast emmory access

LRU-K Replacement Buffer | C/C++, WSL, Git | CS174B

Database buffer using a k-th Least Recently Used replacement strategy, based upon BusTub from Carnegie Mellon University.

  • - Constructed a buffer pool manager to access data in memory, and a replacer to determine which frames to replace
  • - Ensured functions were thread-safe with page locking techniques
  • - Large codebase: referenced the implementation of the disk manager and pages to inform coding decisions
